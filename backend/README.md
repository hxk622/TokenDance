# TokenDance Backend

> FastAPI + Celery | Python 3.11+ | uv

## Quick Start

```bash
# Install dependencies
uv sync --all-extras

# Run dev server
uv run uvicorn app.main:app --reload

# Run tests
uv run pytest tests/
```
