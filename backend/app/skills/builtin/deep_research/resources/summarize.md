# 子技能：内容摘要

当网页内容超过 2000 字时，使用此流程进行摘要，避免 Context 爆炸。

## 适用场景

- 网页内容 > 2000 字
- 需要提取关键信息
- 原文需保留但不入 Context

## 摘要步骤

### Step 1: 识别文章结构
- 标题层级（H1/H2/H3）
- 段落分布
- 图表位置
- 关键数据点

### Step 2: 提取关键句
- 每个主要段落提取 1-2 句核心观点
- 保留所有数字数据
- 保留所有引用来源
- 保留专有名词和术语

### Step 3: 生成摘要
- 控制在 200-500 字
- 保持逻辑连贯
- 标注数据来源

### Step 4: 保留元信息
- 原文链接
- 发布日期
- 作者/来源
- 可信度评分

## 输出格式

```markdown
## 摘要：[文章标题]

**来源**：[域名] | **日期**：[发布日期] | **可信度**：★★★★

### 核心观点
- 观点1：...
- 观点2：...

### 关键数据
- 数据1：... (来源: 原文第X段)
- 数据2：...

### 原文链接
[完整文章](URL)

---
*摘要生成时间：YYYY-MM-DD HH:MM*
```

## 示例

**输入**（假设原文 3000 字）：
```
一篇关于 AI Agent 市场的长文...
```

**输出**：
```markdown
## 摘要：2024 AI Agent 市场分析报告

**来源**：techcrunch.com | **日期**：2024-01-15 | **可信度**：★★★★

### 核心观点
- AI Agent 市场预计 2025 年达到 XX 亿美元规模
- 主要驱动因素包括：企业数字化转型、LLM 技术成熟
- 面临的挑战：数据隐私、可靠性、成本

### 关键数据
- 市场规模：2024 年约 XX 亿美元
- 年增长率：预计 35-40%
- 主要玩家市场份额：OpenAI (25%), Anthropic (15%), Google (12%)

### 原文链接
[完整文章](https://techcrunch.com/article)

---
*摘要生成时间：2024-01-15 10:30*
```

## 注意事项

1. **数据准确性**
   - 数字必须与原文一致
   - 不要推测或补充数据
   - 不确定的数据标注"约"

2. **避免过度压缩**
   - 保留关键论据
   - 不要丢失重要上下文
   - 争议性观点要保留两方

3. **引用规范**
   - 摘要中的数据需标注出处
   - 保留原文中的引用链接
   - 明确标注是摘要而非原文
