# 金融研究场景模板

> 基于 Deep Research Skill 的金融投研专项扩展
> 定位：投研效率工具，辅助信息整合与分析，不提供投资建议

## 场景定位

金融研究是 Deep Research 的**高价值垂直场景**。本模板提供：
- 金融数据源整合策略
- 财务分析框架
- 投研报告模板
- 市场情绪分析

## 使用方式

当用户意图包含以下关键词时，自动加载此模板：
- "股票"、"基金"、"债券"、"期货"
- "财报分析"、"投研"、"估值"
- "市场分析"、"行业研究"
- "宏观经济"、"政策解读"
- 具体股票代码（如 600519、AAPL）

## 金融研究工作流程

### Phase 1: 研究范围界定

```yaml
研究类型:
  个股研究:
    - 基本面分析（财报、业务、管理层）
    - 技术面分析（K线、成交量、指标）
    - 估值分析（PE、PB、DCF）
    - 舆情分析（市场情绪、机构观点）
    
  行业研究:
    - 行业规模与增速
    - 竞争格局
    - 产业链分析
    - 政策影响
    
  宏观研究:
    - 经济指标（GDP、CPI、PMI）
    - 货币政策
    - 财政政策
    - 国际形势
```

### Phase 2: 数据采集策略

#### 2.1 数据源分类

| 数据类型 | 来源 | 优先级 | 备注 |
|----------|------|--------|------|
| **财务数据** | 东方财富、同花顺、Wind | ⭐⭐⭐ | 需验证数据时效性 |
| **公告信息** | 巨潮资讯、交易所官网 | ⭐⭐⭐ | 一手信息源 |
| **研报观点** | 券商研报、雪球 | ⭐⭐ | 注意利益冲突 |
| **舆情数据** | 微博财经、雪球讨论 | ⭐⭐ | 市场情绪参考 |
| **新闻资讯** | 财联社、第一财经 | ⭐⭐ | 实时动态 |
| **国际数据** | Yahoo Finance、Bloomberg | ⭐⭐ | 美股/港股 |

### Phase 3: 财务分析框架

#### 3.1 基本面分析维度

```yaml
财务指标:
  盈利能力:
    - ROE (净资产收益率)
    - ROA (总资产收益率)
    - 毛利率
    - 净利率
    
  成长能力:
    - 营收增速
    - 净利润增速
    - 扣非净利润增速
    
  运营能力:
    - 存货周转率
    - 应收账款周转率
    - 总资产周转率
    
  偿债能力:
    - 资产负债率
    - 流动比率
    - 速动比率
    
  现金流:
    - 经营性现金流
    - 自由现金流
    - 现金流/净利润
```

#### 3.2 估值分析方法

```yaml
相对估值:
  - PE (市盈率): 当前价格 / 每股收益
  - PB (市净率): 当前价格 / 每股净资产
  - PS (市销率): 市值 / 营业收入
  - PEG: PE / 盈利增速
  
行业对比:
  - 与行业平均比较
  - 与历史估值比较
  - 与可比公司比较
  
绝对估值 (参考):
  - DCF 折现现金流
  - DDM 股利折现
  - NAV 净资产价值
```

#### 3.3 技术面分析指标

```yaml
趋势指标:
  - MA (移动平均线): MA5, MA10, MA20, MA60
  - MACD (指数平滑异同移动平均)
  - BOLL (布林带)
  
动量指标:
  - RSI (相对强弱指数)
  - KDJ (随机指标)
  
成交量:
  - 量价关系
  - 换手率
  - 资金流向
```

## 合规要求

### 必须遵守

1. **不提供买卖建议** - 只做信息整合
2. **标注数据来源** - 所有数据需注明出处
3. **免责声明必加** - 每份报告必须包含
4. **时效性标注** - 明确数据采集时间
5. **风险提示必有** - 列出主要风险因素

### 禁止事项

- ❌ 预测具体股价
- ❌ 给出明确买卖点
- ❌ 承诺收益率
- ❌ 使用付费数据源未授权数据
- ❌ 涉及内幕信息
